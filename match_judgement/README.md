# 双人对打模式

该目录公开双人对打模式的主体业务编排。[pipeline.py](pipeline.py)保留视频输入、球场锁定、比赛视角追踪、人体上下文、事件理解、规则判罚、可视化和结果发布的完整调用链。模型推理、球场求解、事件评分和规则状态机由受保护组件注入。

公开Pipeline使用`edge_runtime/component_api.py`定义的能力接口，因此评审方可以阅读完整系统如何协同，也可以接入兼容实现；仓库不包含能直接复现核心精度的模型、参数和算法实现。

私有运行时应接受：

```text
--video <path> [--cloud] [--calibrate-only]
```

输出文件可包括标注视频、轨迹数据、事件记录和判罚摘要，具体字段通过`contracts/task-result.schema.json`对外保持稳定。模型、阈值、映射和判罚实现不属于公开仓库。
