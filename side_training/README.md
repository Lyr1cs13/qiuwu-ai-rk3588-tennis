# 个人训练模式

该目录公开个人训练模式的主体业务编排。[pipeline.py](pipeline.py)保留视频输入、侧视球路追踪、姿态感知、动作时序分析、训练事件组织、可视化和结果发布的完整调用链。模型推理、姿态特征构造、动作分段和训练量化算法由受保护组件注入。

公开Pipeline使用`edge_runtime/component_api.py`定义的能力接口，因此评审方能够审阅双模型如何协同，也可以接入兼容实现；仓库不包含训练资产、生产参数和核心算法实现。

私有运行时应接受：

```text
--video <path> [--cloud]
```

输出文件可包括训练标注视频、轨迹记录和动作摘要，具体字段通过`contracts/task-result.schema.json`对外保持稳定。模型、特征构造、动作分段和训练量化实现不属于公开仓库。
